–ó–∞–≥–æ–ª–æ–≤–æ–∫ —Ñ–∞–π–ª–∞ data.csv –∏–º–µ–µ—Ç –≤–∏–¥:

`user_id,user_name,user_ip,date,page_url`

–î–ª—è –æ–±—Ä–∞–±–æ—Ç–∫–∏ —Ç–∞–∫–æ–≥–æ —Ñ–∞–π–ª–∞ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –ø—Ä–æ–¥–µ–ª–∞—Ç—å —Å–ª–µ–¥—É—é—â–∏–µ —à–∞–≥–∏:

1. –æ—Ç–∫—Ä—ã—Ç—å —Ñ–∞–π–ª –¥–ª—è —á—Ç–µ–Ω–∏—è
2. –ø—Ä–æ—á–∏—Ç–∞—Ç—å –∫–∞–∂–¥—É—é —Å—Ç—Ä–æ–∫—É —Ñ–∞–π–ª–∞
3. —Ä–∞–∑–±–∏—Ç—å –∫–∞–∂–¥—É—é —Å—Ç—Ä–æ–∫—É —á–µ—Ä–µ–∑ —Ä–∞–∑–¥–µ–ª–∏—Ç–µ–ª—å **`,`** –Ω–∞ —Å–ø–∏—Å–æ–∫ –∑–Ω–∞—á–µ–Ω–∏–π
4. –∏–∑–≤–ª–µ—á—å –∏–º–µ–Ω–∞ —Å—Ç–æ–ª–±—Ü–æ–≤
5. –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –∏–º–µ–Ω–∞ —Å—Ç–æ–ª–±—Ü–æ–≤ –∏ —Å–ø–∏—Å–æ–∫ –∑–Ω–∞—á–µ–Ω–∏–π –∏–∑ —Å—Ç—Ä–æ–∫ –¥–ª—è —Å–æ–∑–¥–∞–Ω–∏—è —Å–ª–æ–≤–∞—Ä–µ–π
6. –æ—Ç—Ñ–∏–ª—å—Ç—Ä–æ–≤–∞—Ç—å –Ω–µ –ø–æ–¥—Ö–æ–¥—è—â–∏–µ —Å—Ç—Ä–æ–∫–∏
7. –æ–±—Ä–∞–±–æ—Ç–∞—Ç—å –∏–Ω—Ç–µ—Ä–µ—Å—É—é—â–∏–µ –∑–Ω–∞—á–µ–Ω–∏—è

```python
with open('data.csv', 'r', encoding='utf-8') as file: # –û—Ç–∫—Ä—ã–≤–∞–µ–º —Ñ–∞–π–ª data.csv –¥–ª—è —á—Ç–µ–Ω–∏—è
    file_lines = (line for line in file) # –ß–∏—Ç–∞–µ–º –∫–∞–∂–¥—É—é —Å—Ç—Ä–æ–∫—É —Å –ø–æ–º–æ—â—å—é –≥–µ–Ω–µ—Ä–∞—Ç–æ—Ä–Ω–æ–≥–æ –≤—ã—Ä–∞–∂–µ–Ω–∏—è
    line_values = (line.rstrip().split(',') for line in file_lines) # –∞–∑–±–∏–≤–∞–µ–º —Å—Ç—Ä–æ–∫–∏ —Ñ–∞–π–ª–∞ —á–µ—Ä–µ–∑  —Ä–∞–∑–¥–µ–ª–∏—Ç–µ–ª—å , –Ω–∞ —Å–ø–∏—Å–æ–∫ –∑–Ω–∞—á–µ–Ω–∏–π
    file_headers = next(line_values) # –ò–∑–≤–ª–µ–∫–∞–µ–º –ø–µ—Ä–≤—É—é —Å—Ç—Ä–æ–∫—É, –∫–æ—Ç–æ—Ä–∞—è —è–≤–ª—è–µ—Ç—Å—è –Ω–∞–∑–≤–∞–Ω–∏–µ–º —Å—Ç–æ–ª–±—Ü–æ–≤
    line_dicts = (dict(zip(file_headers, data)) for data in line_values) # –°–æ–µ–¥–∏–Ω—è–µ–º –Ω–∞–∑–≤–∞–Ω–∏—è —Å—Ç–æ–ª–±—Ü–æ–≤ —Å —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–∏–º–∏ –∏–º –∑–Ω–∞—á–µ–Ω–∏—è–º–∏

    # –§–∏–ª—å—Ç—Ä—É–µ–º –Ω–µ–ø–æ–¥—Ö–æ–¥—è—â–∏–µ —Å—Ç—Ä–æ–∫–∏ –∏ –æ—Å—Ç–∞–≤–ª—è–µ–º —Ç–æ–ª—å–∫–æ —Ç–µ, –≤ –∫–æ—Ç–æ—Ä—ã—Ö —Å—Ç–æ–ª–±–µ—Ü page_url —Å–æ–¥–µ—Ä–∂–∏—Ç —Ç–µ–∫—Å—Ç site.ru
    result = (
        line['user_name'], line['user_ip']
        for line in line_dicts
        if 'site.ru' in line['page_url']
        )

    for index, (name, ip) in enumerate(set(result), 1):
        print(f'{index}. {name} --- {ip}')
```
